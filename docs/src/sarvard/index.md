# SÅRVÅRD PÅ AKUTEN

Denna artikelserie går igenom de basala grunderna i sårvård på akuten. Detta innefattar hudbiologi, handläggning, och antibiotika. Slutligen finns även bra mallar för att plocka ihop sutur-/incisionsset samt patientinformation som enkelt kan skrivas ut.
