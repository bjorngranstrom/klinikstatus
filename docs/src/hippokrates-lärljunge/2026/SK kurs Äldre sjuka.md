# <ye>Äldre multisjuka</ye>

Anteckningar från SK kursen "Akut omhändertagande av den multisjuka patienten" på Danderyds sjukhus. Mycket bra och engagerade föreläsare.

### AKUT NJURSVIKT

- Njuren utsöndrar insulin. Klokt att beakta insulindosering så man inte överdoserar.
- Urea generellt dåligt prov då det stiger av många orsaker.
- Aciklovir är rejält nefrotoxiskt, används med försiktighet redan vid mild njursvikt.
- Bactrim hämmar krea-clearance -> kan öka krea 13-35% utan att i verkligheten påverka GFR.
- Relativ GFR utgår från att man väger 70kg. Absolut GFR används således på över/underviktiga.
- Kontrastinducerad nefropati: oklart om detta existerar, i dagsläget kan det vara bra att känna till men ska INTE förhindra viktig kontrastundersökning. Man kan tänka två varv om CKD5.
- Törst korrelerar bra med dehydrering.

### SMÄRTA

- Primärt två smärtmekanismer som intresserar oss: <gr>nociceptiv</gr> och <gr>neuropatisk</gr>. Båda aktiverar det sympatiska nervsystemet och ger således typiska kliniska fynd.
- Toradol är extremt effektivt mot stensmärtor, i synnerhet uretärstenssmärta men också gallstenssmärta. Reducerar smärta genom primärt 2 mekanismer där båda härleds till minskad <bl>prostaglandinsyntes</bl>:

    > TORADOL <re>⊣</re> COX-1 → <re>↓</re> Prostaglandiner → <re>↓</re> Aktivering FP receptor → <re>↓</re> Fosfolipas C → <re>↓</re> Glattmuskelkontraktion → <re>↓</re> Smärta
    
    > TORADOL <re>⊣</re> COX-1 → <re>↓</re> Prostaglandiner → <re>↓</re> Afferent njurarteriolflöde → <re>↓</re> GFR → <re>↓</re> Stassmärta
