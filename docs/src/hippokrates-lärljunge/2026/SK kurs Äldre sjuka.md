# <ye>Äldre multisjuka</ye>

Anteckningar från SK kursen "Akut omhändertagande av den multisjuka patienten" på Danderyds sjukhus. Mycket bra och engagerade föreläsare.

### AKUT NJURSVIKT

- <gr>INSULIN OCH NJURSVIKT:</gr> Njuren utsöndrar insulin. Klokt att beakta insulindosering så man inte överdoserar.
- <gr>UREA:</gr> Urea generellt dåligt prov då det stiger av många orsaker.
- <gr>ACIKLOVIR:</gr> Rejält nefrotoxiskt, används med försiktighet redan vid mild njursvikt.
- <gr> BACTRIM:</gr> Hämmar krea-clearance -> kan öka krea 13-35% utan att i verkligheten påverka GFR.
- <gr>RELATIV GFR:</gr> Utgår från att man väger 70kg. Absolut GFR används således på över/underviktiga.
- <gr>KONTRASTINDUCERAD NEFROPATI:</gr> Oklart om detta existerar, i dagsläget kan det vara bra att känna till men ska INTE förhindra viktig kontrastundersökning. Man kan tänka två varv om CKD5.
- <gr>TÖRST:</gr> Korrelerar bra med dehydrering.
- <gr>HYPERNATREMI:</gr> >170 Na -> 57% dödlighet inhouse.

### SMÄRTA

- <gr>SMÄRTMEKANISMER AV INTRESSE:</gr> <bl>nociceptiv</bl> och <bl>neuropatisk</bl>. Båda aktiverar det sympatiska nervsystemet och ger således typiska kliniska fynd.
- <gr>STENKOLIK</gr>: Toradol är extremt effektivt mot stensmärtor, i synnerhet uretärstenssmärta men också gallstenssmärta. Reducerar smärta genom primärt 2 mekanismer där båda härleds till minskad <bl>prostaglandinsyntes</bl>:

    > TORADOL <re>⊣</re> COX-1 → <re>↓</re> Prostaglandiner → <re>↓</re> Aktivering FP receptor → <re>↓</re> Fosfolipas C → <re>↓</re> Glattmuskelkontraktion → <re>↓</re> Smärta
    
    > TORADOL <re>⊣</re> COX-1 → <re>↓</re> Prostaglandiner → <re>↓</re> Afferent njurarteriolflöde → <re>↓</re> GFR → <re>↓</re> Stas/kapselsmärta

- <gr>AKUT LUMBAGO:</gr> Muskelspasm → muskelrelaxerande → <br>Oxascand 10mg mycket bra (till skillnad från opioid, paraflex, nsaid)</br>
- <gr>ALVEDON VID CANCER:</gr> <br>Om patienten har morfinbehandling tillför alvedon sällan ytterliggare smärtlindring. Överskatta således inte behovet, per os tabletter kan bli mycket mödosamt.</br>
- <gr>NJURSTENSSMÄRTA:</gr><br> 1:a Toradol;</br><br>2:a Robinul 0,2 mg/ml 0,5 - 1 ml iv/im/sc alternativt </br><br>Buscopan 20 mg/ml 1-2 ml iv(ges över 5 minuter)/im/sc.</br>
- <gr>KORTISON:</gr> Kortisonanvädning i 12-14 dagar går att bryta rakt av.
- <gr>COX2 OCH BLÖDNING:</gr> Överväg COX2-hämmare vid ökad blödningsrisk.
- <gr>OPIOIDER:</gr> 
    - Vid behov 1/6 av dygnsdos.
    - Byte från per os -> parenteralt -> halvera dosen!
    - Illamående vanligt; kräkcentrum anpassar sig efter 1-2 veckor; Postafen 25 mg x 2 kan ges.
- <gr>POST-OP SMÄRTA:</gr> Varar som regel inte mer än 10 dagar! Om fortsatt smärta får man tänka om.
- <gr>KRONISK NEURALGI:</gr> Ketanest 2,5-5mg/h.
- <gr>ROBINUL:</gr>Motverkar slembildning men påverkar INTE befintligt slem. Här är det kroppsposition (rescue position) som är det viktigaste.

### ENDOKRINOLOGI

- <gr>OSMOLALITET:</gr> "Mellan tummen och pekfingret"-uträkning -> 2(Na + Glu + Urea)
- <gr>HYPEROSMOLÄRT SYNDROM:</gr> 
    - Läkemedel som höjer glukos eg. kortison
    - Läkemedel som är dehydrerande eg. furix
    - Sänka glukos långsamt pga risk för hjärnödem -> 0,025-0,05 E/kg/h insulin iv; ~1-3 mmol/h
- <gr>DEHYDRERING OCH TROMBOS:</gr> Vid uttalad dehydrering kan det vara klokt att överväga trombosprofylax eg 4500 E fragmin. Högvisköst blod ökar propprisk signifikant.
- <gr>ENDOGEN KORTISON:</gr> Vi producerar ca 5mg prednisolon i ekvivalenta termer fysiologiskt per dag.

### ASPIRATIONSPNEUMONI

- <gr>KLINIK:</gr> ↓SATURATION + ↑ROSSEL + ↑SLEM + SVÄLJSVÅRIGHETER.
- <gr>TIDSFÖNSTER:</gr> 4-6h att utveckla infiltrat radiologiskt basalt/apikalt.
- <gr>ANTIBIOTIKA:</gr> Vänta tills dag 3. NPH- och svalgodling. Typiska agens från munhåla -> Strep, HiF. AB-val således PcG (BensylPC).

### VENÖS TROMBOEMBOLISM

- <gr> ABSOLUT GFR: </gr> Använd på överviktiga -> Många underbehandlas! Stor kroppsvolym -> falskt ↓GFR vid relativ uträkning!
- <gr> NOAK OCH BLÖDNING: </gr> Gör slemhinnor signifikant mer lättblödande! Tag alltid detta i beaktning inför insättning. Exempelvis kan det vara motiverat att använda heparin/waran vid slemhinnecancer/tarmcancer.
- <gr> RISK FÖR VTE MED AK-BEH: </gr> Inga absoluta siffror men risken betraktas som minimal. Undantag pankreasmalignitet.
- <gr> COMPARTMENT: </gr> DVT -> Stasat ben -> AK-behandling -> Ökande smärta och svullnad -> Blödning? -> Ultraljud för att bekräfta -> Hematom? -> JA? -> Compartment -> Akutbehandling
- <gr> "TROMBSPETS KAN INTE VISUALISERAS":</gr>
    - Orsak till att vidareutreda med CT buk:
        - Hög malignitetsmisstanke
        - Kärlkirurgisk intervention
            - Uttalad flegmasia cerulea dolens ("smärtsam blå inflammation") -> massiv venös obstruktion. Bäckenvener dålig kollateral cirkulation. Mycket sällsynt. Detta är vävnadshotande.
            - < 14 dagar
            - < 50 år
    - Dessa patienter behöver inte alltid läggas in.
    - Cavafilter används sällan -> Svårt att ta ut efter 4-6v (växer fast) samt > 50% trombotiserar och går inte att ta ut.
- <gr> LUNGEMBOLI RISK VID DVT, SUBKLINISK OCH SYMPTOMGIVANDE: </gr>
    - Distal 20%
    - Proximal 50%
    - Ovanför inguinalligament OR 2-4
- <gr>SUBDURALHEMATOM:</gr> Farligast vid reblödning -> Svåra att stoppa
- <gr>SEGMENTELL LE:</gr> Betraktas som bifynd i första hand om pat. ej har symptom typiska vid LE.
- <gr>MORTALITET VID HÖFTFRAKTUR:</gr> Dubbleras varje dag man väntar på höftoperation.
- <gr> ARTÄREMBOLI OCH HEPARIN: </gr> Ingen evidens att det minskar artärtromber som vid exempelvis förmaksflimmer.
- <gr> REDUCERAD NOAK: </gr> Dåligt VTE-skydd -> överväg byta till waran/heparin om mycket slemhinneblödningar eg tarmblödning.
- <gr> DDIMER: </gr> 
    - Fibrinogen: trombin+faktor XIII-> Fibrin: Fibrinolys -> Plasmin (bryter ned fibrin)-> D-Dimerfragment  
    - Halveringstid 6-8 timmar.
    - Förhöjd 1-2 veckor efter trombupplösning.
    - Subakut -> Normalisering 2-4 veckor, kan kvarstå 1-3 månader beroende på nedbrytningshastighet samt trombvolym.

- <gr>ELCHOCK:</gr> Man får INTE en stöt av ICD:n trots att man har hud mot hud kontakt när den avger en elektrisk urladdning på patienten.

### AKUT KONFUSION (DELIRIUM)

_Utmärkande för delirium är kort debut (timmar-dagar) med fluktuerande grad över dygnet och blir oftast sämre nattetid. Ofta föreligger en uppmärksamhetsstörning i form av svårighet att hålla fokus. Perceptionsstörning samt agitation/apati förekommer._

- <gr> SYMMETRISK MOTORIK: </gr> Sällan strukturell hjärnskada.
- <gr> PARTIELLT ANFALL: </gr> Delirium kan ibland yttra sig som partiellt anfall. Det är inte fel att ge Keppra, förslagsvis 2 gram då detta inte är skadligt.
- <gr> GRADERING AV VAKENHET: </gr>
    - Vaken: Orienterad, följer uppmaning
    - Somnolens: Trött, somnar utan stimuli
    - Stupor: Svårväckt men går att väcka vid kraftig stimuli
    - Koma: Ej väckbar, vegetativ
- <gr> RADIOLOGI UTAN TYDLIG FRÅGESTÄLLNING: </gr> "Utesluta subduralhematom"
- <gr> FRIKOSTIG MED: </gr> Tiamin 500 mg x 3 i 3 dagar (2-4g tiamin) samt 20 mmol Mg x 1

### PARKINSON'S PÅ AKUTEN

- <gr> TIDSFÖNSTER: </gr> Prioritera dessa patienter då dom snabbt utvecklar rigiditet vid uteblivna doser dopamin. Blir således svårare att häva desto rigidare dom blir samt att dom tappar förmågan att svälja vilket tvingar användning av nasogastrisk sond.
