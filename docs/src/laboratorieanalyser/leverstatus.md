# Leverstatus

::: info Inkluderar Bilirubin, ALAT, ASAT, GGT, ALP, pankreasamylas. Ibland PK/APTT som tillägg.
:::


### Bilirubin

::: info Referensvärden
5-25 (total bilirubin)<br>
Konjugerad <4<br>
[µmol/L] <br>
Intraindividuell variation total bilirubin 22%.<br>
Intraindividuell variation konjugerat bilirubin 37%.
:::

::: tip Farmakokinetik
- Dygnsvariation med morgonvärdet ungefär dubbla eftermiddagsvärdet.
- Okonjugerat bilirubin har en halveringstid på 4 dygn.
- Konjugerat bilirubin faller som regel snabb vid åtgärdandet av gallgångsobstruktion
:::

Nedbrytningsprodukt av erytrocytens hemgrupp. Lipofil, transporteras med albumin som okonjugerat. Blir vattenlösligt i levern som konjugerad form.<br>

<gr>KLINISK ANVÄNDBARHET</gr> 
- Viktigt prov som används i kombination med andra prover för att differentiera mellan gallgångsstas, leversvikt, konjugeringssjukdomar, och hemolys.
- Ikterus kan ses i sclera vid 30-50 och i hud vid 50-70. 

#### <gr>TOLKNINGAR:</gr> 

⇧ <re>KONJUGERAT</re>  <br>
- OBSTRUKTION I GALLVÄGAR
- LEVERSKADA (Cirros, alkohol, paracetamol, steatos)
- HEPATIT (HeA+E, EBV) 
- HÖGERKAMMARSVIKT (dålig prognos vid höga värden)
- SEPSIS (Gram- pga endotoxiner. Dålig prognos)
- POSTOPERATIV IKTERUS (absoprtion av stort hematom)
- PRIMÄR BILIÄR CIRROS<br>

⇧ <re>OKONJUGERAT</re>  
- HEMOLYS
- GILBERTS SYNDROM<br>
- RIFAMPICIN+PROBENECID 

#### <gr> LÄKEMEDELSUTLÖST IKTERUS TILL FÖLJD AV LEVERTOXICITET </gr>
- Sällsynt men allvarligt och måste utredas skyndsamt!
- Delvis kolestatisk bild
- <re>LÄKEMEDEL:</re> AMOXICILLIN, FLUKLOXACILLIN, IBUPROFEN, FENYTOIN, KARBAMAZEPIN, DIAZEPAM, FLUOXETIN, CYTOSTATIKA, HEPARIN, AMIODARON, CYKLOSPORIN, ÖSTROGENER, ANABOLA STEROIDER
